测试种类

React App 的测试主要由单元测试（Unit Test），快照测试（Snapshot Test）和端到端测试（E2E Tets）组成。

如测试金字塔所示，你应该对每个组件进行多个单元测试，对每个组件进行一次或两次快照测试，以及测试链接在一起的多个组件的一次或两次端到端测试。
